定义：

除非本协议另有规定，以下词语在本协议中的定义如下：

1.甲方：是指通过丙方平台注册账户的会员，承诺其具有一定的投资风险识别能力和投资风险承担能力；并承诺对本协议涉及的投资资金具有完全的支配能力，该投资资金是其自有闲散资金，为其合法所得；并承诺其提供给丙方的信息是完全真实及有效的。

2.乙方为已在丙方平台注册，并承诺其提供给丙方的信息是真实有效的自然人或符合丙方审核的机构组织。

3.丙方：是一家在杭州市合法成立并有效存续的有限责任公司，拥有跟操盘平台（含tg.simutz.com网站、微信公众号、手机APP等）的经营权，从事提供信息咨询、为交易提供信息中介服务等业务。

4.合作机构：与丙方建立合作关系的机构，包括但不限于：第三方支付机构（包括但不限于腾讯支付及其关联公司）、开户通道等。

以上各方本着实事求是、平等互利、共同发展的原则，就丙方在跟操盘平台上推出 江浩交易策略（以下简称“江浩策略”），乙方作为操盘人发起【江浩实盘策略】进行期货投资，甲方作为乙方期货投资跟投的合买人进行合买的相关事宜，经过三方友好协商，达成协议如下：

一、本期策略意向

1.1 丙方在其跟操盘平台推出“ 期货收益类合买计划 ”（以下简称“跟操盘”）。本期跟操盘是由合格操盘人发起、投资人合买的投资理财计划。

1.2 乙方承诺，其已根据丙方要求提供相关资料，由丙方进行相应筛选，经过专业审核后成为平台的合格操盘人。乙方作为操盘人发起【江浩实盘策略】计划（以下简称“江浩策略”或“本策略”），投资人通过跟踪复制本策略进行期货投资，在投资业绩达到收益分成线时获得合买收益分成，在策略亏损时自行承担投资风险。

1.3 甲方授权丙方依其平台制定的规则审核操盘人，并授权丙方挑选乙方作为合格操盘人。甲方愿意作为投资人对乙方发起的本期策略进行跟投，并承诺当跟投账户盈利按约向丙方支付分红，自愿承担跟投策略产生的投资风险。

1.4 乙方同意，为控制合买的风险，授权丙方对本策略设置相应的止损比例和止损线，并对乙方的投资操作进行实时监控，以便及时采取强行平仓等风险控制措施。乙方认可丙方基于以上授权而采取的相应措施及后果，并愿意承担由此可能产生的损失。

1.5甲乙方知晓并同意， 丙方未以任何方式对本策略的投入资金、可获收益做出任何承诺、保证。甲方及乙方投入的资金可能面临遭受一定损失的风险。

2.2 策略初始单位净值为1.00，分红线为1.3 。

2.3 甲乙双方一致同意，丙方可根据甲乙双方资金不同比列随时调整跟投倍数。

2.6 收益分成线：在策略到期或清算时，当本策略获得超额收益的，甲方应支付跟投超额收益的30 %予丙方，作为合买收益分成，再由丙方根据相关税务规定扣除税率后支付给乙操盘手。收益分成的具体计算方式请见本协议“六、本策略的清算与收益分配”部分及跟投详情页。

三、风险控制

3.1 为引导乙方有效控制策略风险，甲乙双方同意并授权丙方平台对策略账户从投资范围、持仓集中度、止损线、禁止违规操作等几个方面进行约束和管理。

3.2 投资范围限制策略的投资范围：靠近或在涨跌停板逆势开仓，交易冷门合约，以及其他违反交易规则的相关行为。

3.3 持仓集中度限制

3.4 止损线

3.4.1 丙方将实时监控乙方操盘的风险情况，当跟投穿仓比跌至操盘手历史最大回撤时，丙方平台有权提醒并限制乙方交易，以协助控制风险，维护甲乙双方相应权益。

3.4.2 策略单位净值触及止损线导致丙方平台进行强制平仓的，丙方平台有权代甲方卖出策略账户的全部持仓，本期策略提前终止。

3.4.3 在上述强行平仓过程中，丙方可以任何可能成交的价格，包括跌停价，卖出策略账户内的持仓。同时，丙方不保证在止损线完全卖出，由于行情波动以及操盘难度造成账户净值跌破止损线的，甲乙双方无异议。若平仓后，价格上涨，造成的损失由甲方和乙方自行承担，丙方对此不承担任何责任。

3.5 禁止违规操作

3.5.1 乙方不得进行操纵市场、对敲、接盘、内幕交易、利益输送等违反股票、期货等交易法律法规及证券、期货等公司规定的交易。

3.5.3 乙方违反前款约定进行违规交易被丙方知悉的，丙方有权采取相应措施，包括但不限于限制或禁止乙方对跟投账户的操作，强平清仓并提前终止本合约，要求乙方支付违约金并承担因其违规交易给甲方、丙方造成的直接或间接损失等。

3.6 其他约定

3.6.1 除上述风险限制外，丙方及合作的期货公司等都有可能会设定总体持仓限制或交易限制，乙方持仓和交易也需满足此类限制，如因触犯某些限制导致乙方未能下单或下单未能成交，乙方和甲方表示谅解，丙方不因此承担任何责任。

3.6.2 丙方可根据风控需求和市场变动，对以上风控规则进行修订并在丙方网站发布。甲乙双方需应不时地注意丙方及具体规则的变更，若甲乙双方在本协议及具体规则内容公告变更后继续使用本服务的，表示甲乙双方已充分阅读、理解并接受修改后的协议和具体规则内容，也将遵循修改后的协议和具体规则使用本网站的服务；同时就甲乙双方在协议和具体规则修订前通过本网站进行的交易及其效力，视为已同意并已按照本协议及有关规则进行了相应的授权和追认。

四、本策略的估值与净值计算

4.1 为客观、准确的反映账户资产的价值，甲方使用本人名义开立期货交易账户，并注册期货保证金监控中心，系统会自动进行净值计算,并以网站公布实际盈利金额为准。甲方和乙方认可丙方的净值计算和利润数据，并同意以此作为清算和收益分配时的计价依据。

五、本策略的提前终止

5.1 本合约在以下情况下提前终止：

5.1.1 当跟投账户单位净值涨幅大于等于可提前终止盈利比，且账户全部为现金资产，乙方可向丙方发起提前终止申请，并清算，并获得丙方批准，后续继续操作则合约自动续期顺延；

5.1.2 当跟投账户单位净值跌至止损线，丙方有权对跟投账户完成强制平仓的；

5.1.3 因乙方违反本协议第3.5.1条的约定从事违规交易，被丙方知悉而采取提前终止跟投账户的处置措施的；

5.1.4 因乙方违规操作、个人债务等原因，导致乙方的指定账户（包括银行账户和跟投账户）被有权机关依法采取查封、冻结、划拨等强制措施的；

5.1.5 因其他原因，各方协商提前终止本策略的。

5.2 合约提前终止的，由乙方自行或丙方平台代乙方对跟投账户进行强制平仓。跟投账户持仓清仓的完成当日为跟投账户的提前终止日。

六、本策略的清算与收益分配

6.1 在策略单位净值未触及平仓的情况下，丙方于策略到期时进行清算和收益分配；在跟投账户单位净值触及止损线时，丙方根据本协议第3.4条的约定对跟投账户进行强制平仓操作，则合约提前终止，进行提前清算。

6.2 跟投账户清算前，乙方须将跟投账户内的所有持仓全部卖出；因强制平仓而提前清算的，由丙方平台代乙方将甲方所有持仓全部卖出。

6.3 因合约到期而进行清算的，丙方平台对跟投账户进行清算，甲方向丙方支付利润分红，并由乙方根据相关税务规定，扣除税率后支付给乙方。

6.4收益分配比列为，甲方获得个人账户投资收益的70%，乙方获得利润分配的30%，丙方根据导师服务协议获取平台中间费用，甲方不承担此费用。

策略共享概述及产品风险说明书

一、跟操盘共享策略概述

跟操盘共享策略，是指由合格操盘手/牛人（下文简称“操盘手”或“牛人”）发起，投资人参与的安全便捷的期货投资收益类共享模式。跟操盘共享策略，为广大投资人提供参与期货投资的机会并享受相应的投资收益，牛人通过发布自己的交易策略，并邀请投资人参与，投资者绑定操盘者账户后，即可复制操盘者的操盘结果，双方达到一定业绩回报后获得相应的投资收益和收益分成，其中牛人展示的实盘账户均是真实资金。

所有合格牛人均由平台依据审慎的筛选原则，按照专业的评审方式选出。同时为控制投资人的风险，平台设置相应的止损线，并实时监控，及时采取相关风险控制措施。

跟操盘共享策略在操盘收益达到一定业绩回报时，牛人向投资人收取收益分成，共享收益；如果发生损失，损失在止损比例以内的，风险均由投资人自行承担。

跟操盘共享策略运行中，跟投账户的单位资产净值首次达到止损线时，平台有权对跟投账户强行平仓并提前结算。

风险提示：

XX平台在跟操盘共享策略的投资中仅作为平台提供机构及风险控制机构，仅能保证平台上操盘手或牛人发布的交易策略等信息的真实性，不保证投资人的投资收益。

投资人在加入本组合之前，需明确投资可能获得较高的收益，同时也需要承担相应的投资风险，请仔细阅跟操盘共享策略说明书及操策略跟投产品协议，全面认识本组合合买的风险收益特征和跟投合买特性，并充分考虑自身的风险承受能力和风险偏好，理性判断市场，谨慎做出投资决策。

二、组合要素说明

1.投资范围

大宗商品期货，金融期货。

2.合格操盘手/牛人

拥有优秀的历史投资业绩和风险管理能力且通过平台审核的操盘牛人。

3.适格投资人

具备一定风险投资识别能力和风险投资承受能力，并追求期货收益的投资人。

4.名词释义

在本说明书中，除非文意另有所指，下列词语或简称具有如下含义：

跟操盘共享策略： 复制牛人操盘策略，不用看盘尽享收益。

期货保证金监控中心：主管部门是中国证监会，期货保证金安全存管机构。可向期货公司获取监控中心编码，平台根据结算数据统计净值结果。

清盘线：假设清盘线为0.7，则最大亏损为30%

单位净值：1.2单位净值为收益20%。

备注：（出入金会造成净值计算偏差）

操盘手账户/投资人账户：投资人可绑定操盘手账户设定跟投倍数，实现于操盘手账户收益曲线相近的投资方式

分红清算：当收益超过约定分红线，随时可清算。

成立条件：投资人投资总额原则上不低于操盘手账户资金，或者不少于一半资金（2/1开仓）。

收益分成：账户盈利金额分红。

例：假如跟投账户盈利30万，收益分成如下：

投资人收益＝300000*0.7=210000

操盘人收益=30000*0.3=90000

亏损比例：当组合出现亏损时，协议内亏损由投资人承担。

穿仓：当账户超过约定清盘线，由三方协商解决

预警线：假设清盘线为0.8，则预警线为0.85以此类推，上浮5%作为预警策略调整的风控线。

可提前终止盈利比例：当盈利达到投资者收益预期，而操盘手策略未平仓时，可向平台提申请提前分润

起始日：跟投账户开始运行首日。

终止日：终止合作日期

三、费用及收益分成

开户费：开户费为0。目前平台暂不收取开户费。

手续费：期货交易手续费所有账户默认交易所1.1倍

管理费：赢利达到分红线后，总收益收取30%的管理费由平台代收，平台扣除税率和导师服务费用后，将佣金支付给操盘人

退出费：用户终止合作后(在平台下架)，或终止委托自行交易，除盈利分红外，无产生收益的退出都不收取费用，并可以继续使用自己开立的期货交易账户，后续操作与平台无关

四、申请说明

1. 资质审核

平台基于牛人申请者的实盘信息，自动生成牛人交易报告，并进行资质审核，通过审核者成为平台牛人参与策略共享，更可以发起策略共享理财。

2. 申请期

牛人申请为净值生成后经过团队沟通三个工作日内可上线分享策略。投资者申请则在充分了解风险并资金到账后开始成立

3. 其他

建议在牛人回撤后投资，不建议投资者总在牛人策略高点介入。

五、净值计算

实际盈利分红以累计收益为准，利润提取后排行榜仍会连续更新净值。

六、风险管理

为引导牛人有效控制账户风险，平台会对组合账户从投资范围、持仓集中度、风险实时监控、禁止违规操作等几个方面进行约束和管理。

1. 投资范围限制

牛人的投资范围为大宗商品期货和金融期货，不逆势在涨跌停板开仓。

2. 风险实时监控

平台会实时更新牛人最新净值，品种盈亏，每周盈亏，每月盈亏等数据，以有效提示牛人风险，控制投资人的投资风险。

（1）预警线

预警线为0.8，平台将采取减仓措施，控制投资人投资风险。

（2）止损线

策略止损线为0.7的，盘中或者收盘任意时间段净值接近或者达到止损线时，采取平仓措施，以控制风险

3. 禁止违规操作

牛人不得进行操纵市场、对敲、接盘、内幕交易、洗钱等违反期货交易法律法规及相关法律规定 。
